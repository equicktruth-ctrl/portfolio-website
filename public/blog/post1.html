How I Built My Portfolio

Published: October 29, 2025

Step 1: Identify Goals and Scope

Before starting, I clearly defined what the portfolio needed to accomplish: showcase my operational leadership and IT skills, include certifications and experience details, and allow easy contact. Identifying this focus helped me keep the build lean and purposeful.

Step 2: Selecting Tools and Hosting

I chose technology based on simplicity, scalability, and cost-effectiveness. Visual Studio Code offered powerful editing and Git integration. GitHub managed version control. Netlify enabled simple, automated deployments from GitHub with free SSL and global CDN.

Step 3: File Organization and Naming

Creating a clean folder structure was vital. I used a public/assets folder for images and simple root-level HTML files for structure. This prevented path errors and reduced deployment confusion. Consistent lowercase naming with hyphens was used for all files to avoid case sensitivity problems.

Step 4: Building the Homepage

The homepage contains a strong header with blurbs on my brand statement, experience, skills, and certifications. I added contact info and a downloadable resume, styled with CSS gradients and shadows for polish.

Step 6: Managing Static Blog Posts

Blog posts are individual HTML pages placed in a public/blog folder. Each article has its own well-structured page linked from the homepage. This manual approach is ideal for now, giving me control and reducing complexity without a CMS.

Step 7: Testing and Debugging

  • Multiple hard refreshes and cache clears ensured latest changes appeared.
  • Verifying image paths fixed frequent display issues.
  • Careful Git commits and pushes avoided deployment errors.
  • Netlify deploy logs were reviewed for build status and errors.

Step 8: Future Improvements

  • Adding multilingual support.
  • Migrating to a static site generator like Jekyll or Hugo for easier blogging.
  • Integrating analytics and SEO enhancements.
  • Expanding interactivity with JavaScript features.
